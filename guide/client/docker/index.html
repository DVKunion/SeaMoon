# Docker

推荐使用 docker 一键部署client

$ docker run -it -d --net=host dvkunion/seamoon-client

默认使用--net=host模式，然后浏览器打开：

localhost:7777

即可开启客户端, 点击配置管理

添加云端配置

注意，此处填写的值为 Step.2 中输出的 domain字段，并去除 http:// 协议头，属于对应域名即可。

开启socks5代理，并保持默认:1080不改变。
