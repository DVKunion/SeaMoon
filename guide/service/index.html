    服务

    # 服务

    # 创建服务

    点击 [新增]:

    • 代理名称：用于在平台区分不同的代理
    • 监听协议: 选择代理的服务类型
    • 监听地址: 监听的 IP 信息
    • 监听端口: 监听的端口信息

    # 从账户直接创建服务

    直接从账户创建，将会从选择的账户创建一个新的函数与服务绑定。具体请参考 创建函数实例 (opens new window)

    # 选择已有函数进行关联

    选择已有函数进行关联时，将会从已有的函数实例进行选择，无需进行其他配置。

    # 修改服务

    目前允许修改的内容有：代理类型、监听地址、监听端口三个选项。点击对应服务，即可进行详情修改。

    # 测速

    点击测速，将会通过当前服务访问测速网站进行上行与下行的速率测试。

    # 停止服务

    点击停止服务，代理服务将会停止，但数据不会删除。

    # 删除服务

    点击删除服务，将会删除对应的代理服务数据。
